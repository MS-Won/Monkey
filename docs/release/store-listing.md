# Google Play 스토어 등록 문구 (초안)

> 실제 등록 시 Play Console > 스토어 등록정보에 붙여넣기. 문구/길이는 자유롭게 수정.

## 기본
- **앱 이름(제목, 최대 30자)**: `Monkey - 꿈 해몽`
- **짧은 설명(최대 80자)**:
  `전통 해몽으로 매일 밤 꿈을 기록하고, 아르누보 카드로 오늘의 상징을 만나보세요.`

## 자세한 설명(최대 4000자)
```
Monkey는 당신의 꿈을 기록하고, 오래된 전통 해몽의 시선으로 풀어주는 앱입니다.

■ 전통 해몽에 뿌리를 둔 해석
서양식 상징 풀이가 아닌, 한국 전통 해몽을 근거로 꿈의 상징을 해석합니다.
꿈에 나온 상징(뱀·돼지·물·불 등)을 전통 사전과 맞춰, 나직한 점술가의 어조로 들려드립니다.

■ 아르누보 꿈 카드
해석과 함께 그 꿈에 어울리는 30종의 아르누보풍 여신 카드 한 장을 뽑아드립니다.
꿈기록 탭에서 카드를 넘겨보고, 탭하면 그날의 해몽이 펼쳐집니다.

■ 손쉬운 기록
- 텍스트로 적거나 음성으로 말해 꿈을 기록
- 꿈 기록·해몽·카드가 자동 저장
- 통계 탭에서 최근 자주 나타난 상징과 흐름을 확인

■ 프라이버시
- 계정 없이 사용, 꿈 기록은 기기 안에만 저장됩니다.
- 해석 생성을 위해 꿈 텍스트만 안전하게 처리됩니다.

※ 해몽과 '행운력' 콘텐츠는 전통과 재미를 위한 참고용이며, 미래를 단정하는 예언이 아닙니다.
```

## 그래픽 자료(별도 준비 필요)
- **앱 아이콘**: 512×512 PNG(32비트, 알파). 현 런처 아이콘 기반으로 제작.
- **피처 그래픽**: 1024×500 PNG/JPG (필수).
- **폰 스크린샷**: 최소 2장(2~8장 권장). 홈/해몽결과/꿈기록 카드/통계 화면 캡처 권장. (에뮬레이터에서 캡처 가능)
- (선택) 태블릿 스크린샷.

## 데이터 보안(Data safety) 양식 답변 가이드
- **데이터 수집**: 예 — "앱 활동 / 기타 사용자 콘텐츠"(꿈 텍스트). 해석 생성 목적으로 제3자(OpenAI)에 전송.
- **데이터 공유**: OpenAI로 처리 목적 전송(광고/판매 아님).
- **기기 외 저장**: 사용자의 꿈 기록은 서버에 저장하지 않음(기기 로컬). 처리 후 보관하지 않음.
- **계정/식별자**: 없음.
- **암호화 전송**: 예(HTTPS).
- **삭제 요청**: 앱 내에서 개별 기록 삭제 가능, 앱 삭제 시 전체 삭제.
- **마이크**: 음성 입력용(녹음 미저장, 텍스트만 사용).

## 콘텐츠 등급 / 타깃
- 콘텐츠 등급 설문: 폭력/성적 콘텐츠 없음. (점술/오컬트 관련 문항 정직히 답변)
- 타깃 연령: 성인/일반(만 13세 이상 권장). 아동 대상 아님.
